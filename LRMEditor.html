LearningResource (schema.org)

Name (Title)
? Keep the title short. Use 'Description' for a detailed characterization of your resource.
 (mandatory) 
LearningResourceType
? Select the predominant type or kind characterizing the learning resource (e.g. 'worksheet', 'experiment', 'lesson plan).
 (mandatory) 
Description
? Describe the topic and the intention of the resource.
Note: Do not use quotation marks (") in the description, use apostrophe (') instead!
 (mandatory) 
Author
? The author's name.
 (mandatory) 
Language
? The predominant language of the resource.
 (mandatory) 
License
? Select the appropriate Creative Commons license.
 (mandatory) 
URL
? The Web address where the resource can be found (e.g. https://www.oercommons.org/courses/6-3-weather-climate-water-cycling).
 (mandatory) 


Audience
? For whom is the resource suitable? Teachers? Learners? Experts?
 (optional) 
InteractivityType
? Does the resource expose content, activate the learners or both?
 (optional) 
Keyword(s)
? The keyword(s) associated with the resource (not necessarily bound to a standard vocabulary).
Use the comma (,) as separator!
 (optional) 
Contributor
? The name of a contributor (if applicable).
Klick the 'plus' sign for additional fields (max. 2).
 (optional)   ➕
Reference  (URL) 
? The URL of the resource this resource is based on (e.g. https://www.merlot.org/merlot/viewMaterial.htm?id=773401860).
 (optional) 
Creation Date
? The date the resource has been created (format: yyyy-mm-dd).
 (optional) 
Modification Date
? The date of the last modification of the resource (format: yyyy-mm-dd).
 (optional)
